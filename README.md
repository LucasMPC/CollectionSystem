## Objetivo do Software

O **Collection System** é um sistema desktop desenvolvido para auxiliar colecionadores na organização e gerenciamento de seus itens físicos e digitais, com foco inicial em coleções de jogos de videogame. O sistema visa substituir planilhas manuais por uma interface visual intuitiva ("Gamer Dark") e centralizada.

## Funcionalidades do Sistema (Requisitos)

* **Gerenciamento de Acesso:** Login seguro de usuários.
* **Dashboard:** Visão geral com estatísticas da coleção (total de itens, categorias).
* **Gerenciamento de Coleções:** Criação e visualização de múltiplas coleções (ex: Jogos de PC, Jogos de PS4).
* **CRUD de Jogos:** Cadastro, listagem, edição e exclusão de jogos.
* **Categorização:** Organização por gênero, ano, desenvolvedora e tipo de mídia.
